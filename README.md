## Installation

### Clone this repository using:
<script>
  git clone https://github.com/MikolajFrycz/sweet-dwl-dotfiles
</script>
### Open the cloned folder and make setup.sh executable:
<script>
  cd sweet-dwl-dotfiles
  sudo chmod +x setup.sh
</script>

### run setup.sh as root:
<script>
  sudo ./setup.sh
</script>
